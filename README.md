# Focus Caravaning Dashboard

Dieses Verzeichnis ist ein statisches Dashboard für GitHub Pages oder eine bestehende Website.

## Dateien

- `index.html`: Dashboard-Seite
- `style.css`: Layout und Gestaltung
- `app.js`: Berechnungen und Charts
- `data/kennzahlen.json`: Monatsdaten aus Excel

## Monatliches Update

1. Neuen Monatswert in der Excel-Datei ergänzen.
2. `data/kennzahlen.json` neu erzeugen oder ersetzen.
3. Änderungen zu GitHub hochladen.

Das Dashboard berechnet Monatsvergleich, kumulierte Werte, rollierende 12 Monate und Trendwerte automatisch.

## GitHub Pages

Das komplette Verzeichnis kann in ein GitHub-Repository hochgeladen werden. In den Repository-Einstellungen dann GitHub Pages aktivieren und als Quelle den Branch mit diesen Dateien wählen.
